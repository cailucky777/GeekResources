# Portfolio

> "The best way to get opportunities is to already be doing the thing."

A portfolio is the single most useful artifact a student programmer can build. It's the evidence — for college admissions, internships, scholarships, and your future self — that you do what you say you do.

## The Minimum Portfolio

1. A GitHub profile with a real avatar and a profile README.
2. **3 projects** — each with a screenshot, a clear description, and working code.
3. A one-page personal website with links to the above.

That's it. A recruiter or admissions reader spends ~2 minutes on you. Make it easy for them.
