# Xcode Shortcuts

Learning these pays itself back within a week.

---

## The Essentials

| Shortcut | Action |
|---|---|
| ⌘R | Run |
| ⌘. | Stop |
| ⌘B | Build |
| ⇧⌘K | Clean build folder |
| ⌘U | Run tests |
| ⌘O | Open Quickly (fuzzy file finder) |
| ⇧⌘O | Same, in a new window |
| ⌃⌘↑ | Jump to header / alternate file |
| ⌘\\ | Toggle breakpoint |

## Navigation

| Shortcut | Action |
|---|---|
| ⌘1 | Project navigator |
| ⌘6 | Symbol navigator |
| ⌘7 | Test navigator |
| ⌘8 | Breakpoint navigator |
| ⌘9 | Report navigator |
| ⌘0 | Toggle left sidebar |
| ⌥⌘0 | Toggle right inspector |
| ⇧⌘Y | Toggle debug area |
| ⌃⌘F | Full screen |

## Editor

| Shortcut | Action |
|---|---|
| ⌃I | Reformat selection |
| ⌥⌘← / → | Fold / unfold code |
| ⌥-click | Jump to definition |
| ⌘L | Jump to line number |
| ⌃6 | Open symbol menu in current file |
| ⌘/ | Toggle comment |
| ⌥⌘← | Navigate backward (like browser) |
| ⌥⌘→ | Navigate forward |

## Find

| Shortcut | Action |
|---|---|
| ⌘F | Find in file |
| ⇧⌘F | Find in project |
| ⌃⌘F | Find & replace |
| ⌘E | Use current selection as search term |

## Documentation & Suggestions

| Shortcut | Action |
|---|---|

| ⎋ (Esc) | Show autocomplete |
| ⌃⌘Space | Show emoji/symbols picker |

## Simulator

| Shortcut | Action |
|---|---|
| ⌘⇧H | Home |
| ⌘⇧A | Toggle appearance (light/dark) |
| ⌘K | Toggle software keyboard |
| ⌘⇧R | Rotate |
| ⌘⇧S | Save screenshot |

## Tips

- **Memorize 5 shortcuts per week.** Don't try to learn all at once.
- **`⌘O` replaces the Project Navigator** for most file navigation.
- **`⌃6` is the fastest way to jump within a file** you're already in.
- **Disable "Automatically save on build"** if it's creating friction — then use ⌘S habitually.
