This unit walks a student project all the way to real users' phones — via **TestFlight** (for friends and family) and the **App Store** (for the world).

## Cost & Account Notes

- **Free Apple ID** → Run on your own device, up to 7 days before re-signing.
- **Apple Developer Program** ($99/year) → TestFlight, App Store, more devices, no 7-day limit.
- **Schools** can often apply for the **Apple Developer Program for Education** and get free access. Ask your teacher.

---

## Recommended Duration

**1–2 weeks**, usually alongside a capstone project.
