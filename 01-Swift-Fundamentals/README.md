# Swift Fundamentals - Curriculum Overview

## Module Description

This 10-week module introduces students to the Swift programming language from the ground up. Students will learn core language concepts including variables, control flow, functions, collections, and error handling. By the end of this module, students will be able to write well-structured Swift programs and will be prepared to move on to iOS app development.

## Prerequisites

- Basic computer skills (file management, text editing, keyboard proficiency)
- Ability to install and open Xcode or Swift Playgrounds on a Mac/iPad
- No prior programming experience required

## 10-Week Lesson Plan

### Week 1: Variables, Constants, and Data Types
**Learning Objectives:**
- Distinguish between variables (`var`) and constants (`let`)
- Use fundamental data types: `Int`, `Double`, `String`, `Bool`
- Apply string interpolation to build dynamic strings
- Perform basic arithmetic operations

### Week 2: Control Flow
**Learning Objectives:**
- Write conditional logic using `if`, `else if`, and `else`
- Use `switch` statements for multi-case branching
- Iterate with `for-in` loops over ranges and collections
- Apply `while` and `repeat-while` loops appropriately

### Week 3: Functions and Closures
**Learning Objectives:**
- Define functions with parameters and return values
- Use argument labels and default parameter values
- Understand scope and the difference between local and global variables
- Write simple closures and use trailing closure syntax

### Week 4: Collections
**Learning Objectives:**
- Create and manipulate `Array`, `Dictionary`, and `Set` types
- Iterate over collections using `for-in` loops
- Apply common collection methods (`append`, `remove`, `contains`, `count`)
- Choose the appropriate collection type for a given problem

### Week 5: Optionals and Error Handling
**Learning Objectives:**
- Explain why optionals exist in Swift and what problem they solve
- Safely unwrap optionals using `if let`, `guard let`, and nil coalescing (`??`)
- Define and throw custom errors using `enum` and `Error` protocol
- Handle errors with `do-catch` blocks

### Week 6: Structs and Classes
**Learning Objectives:**
- Define custom types using `struct` and `class`
- Add properties and methods to custom types
- Understand value types vs. reference types
- Use initializers to set up instances

### Week 7: Protocols and Extensions
**Learning Objectives:**
- Define protocols to establish contracts for types
- Adopt and conform to protocols
- Extend existing types with new functionality using `extension`
- Use protocol-oriented design to write flexible code

### Week 8: Enumerations and Pattern Matching
**Learning Objectives:**
- Create enums with raw values and associated values
- Use pattern matching with `switch` on enums
- Apply `CaseIterable` and `Codable` to enums
- Model real-world states using enumerations

### Week 9: Generics and Higher-Order Functions
**Learning Objectives:**
- Write generic functions and types
- Use `map`, `filter`, `reduce`, and `compactMap` on collections
- Chain higher-order functions for data transformations
- Understand type constraints on generics

### Week 10: Review, Mini-Project, and Assessment
**Learning Objectives:**
- Synthesize all concepts from Weeks 1-9
- Plan, build, and present a small command-line Swift project
- Demonstrate debugging and problem-solving skills
- Reflect on personal growth and identify areas for further study

## Assessment Methods

| Method | Weight | Description |
|---|---|---|
| Weekly Exercises | 30% | Graded problem sets assigned each week |
| In-Class Participation | 10% | Engagement in warm-ups, live coding, and discussions |
| Midterm Quiz (Week 5) | 15% | Written/code quiz covering Weeks 1-5 |
| Demo Code Reviews | 15% | Peer review of demo extensions and modifications |
| Final Mini-Project | 30% | Individual command-line project presented in Week 10 |
