# Why Learn a Second Language

You're learning Swift. Why bother with Python or JavaScript?

---

## It Makes You a Better Swift Programmer

Every language emphasizes different concepts. Python teaches readability and simplicity. JavaScript teaches you how flexible (and dangerous) dynamic typing is. After a month of Python, Swift's type system feels like a feature rather than an obstacle.

---

## Different Tools for Different Jobs

- **One-off data cleanup** — Python with pandas in 10 lines.
- **Quick web scrape** — Python with BeautifulSoup.
- **Landing page for your app** — HTML + CSS, maybe a sprinkle of JavaScript.
- **Small API your iOS app calls** — Python (Flask/FastAPI) or JavaScript (Express) in a few files.
- **Game prototype** — Unity uses C# (already in this curriculum).

Forcing yourself to do all of these in Swift is possible but awkward.

---

## Python: Why It's Valuable

- **It's everywhere.** Data science, ML, automation, backend, scripting, research.
- **Readable** — Python code almost reads like English.
- **Huge ecosystem** — `pip install` almost anything.
- **First-class AI/ML ecosystem** (PyTorch, scikit-learn, Hugging Face).

Great second language for anyone learning Swift.

---

## Web (HTML/CSS/JS): Why It's Valuable

- **You need it to promote your apps.** Landing pages, docs, portfolios.
- **The most accessible developer platform** — anyone with a browser can see your work.
- **Foundation of many "native-feeling" frameworks** — React Native, Electron, Ionic.

Even if you don't plan on being a web developer, the ability to publish something on the internet in an afternoon is non-negotiable.

---

## How to Add a Second Language Without Drowning

1. **Pick one project** you'd do in the new language. Don't try to "learn Python" in the abstract.
2. **Write crappy code for a week.** You'll google everything. That's normal.
3. **Get comfortable with the ecosystem's core tools** — for Python: `pip`, `venv`. For web: VS Code, the browser's devtools.
4. **Return to Swift.** Notice what you'd borrow from the other language. That's growth.

---

## Suggested First Projects

| Language | Starter Project |
|---|---|
| **Python** | Script to scrape your class schedule from a website and export to CSV |
| **Python** | Small Flask server that returns a joke from a list (iOS app consumes it) |
| **JavaScript** | A one-page site for your iOS app with screenshots + download link |
| **JavaScript** | A Chrome extension that highlights certain words on web pages |
| **Python + JS** | A web-based version of your iOS app (e.g., flashcards) using Flask + a simple front-end |

Pick one. Ship it in 2 weeks. You've expanded your range.
