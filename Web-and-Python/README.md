# Web and Python

> Even if your main path is iOS, knowing one scripting language and the basics of the web will **multiply** what you can build.

This is not a full course — just the practical "you should be able to do this" level so you can:

- Write a quick script to clean up a CSV.
- Build a small web page to promote your app.
- Run a tiny backend for your iOS project.
