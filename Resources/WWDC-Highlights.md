# WWDC Highlights for Educators

Key WWDC (Apple Worldwide Developers Conference) sessions that are especially relevant for K12 CS teachers. Sessions are organized by topic and include links to Apple's developer site where videos are freely available.

All sessions can be viewed at [developer.apple.com/videos](https://developer.apple.com/videos/).

---

## SwiftUI Essentials

| Session | Year | Description | Why It Matters for Teaching |
|---------|------|-------------|-----------------------------|
| *SwiftUI essentials* | 2024 | Overview of SwiftUI's core concepts, views, modifiers, and state management. | Provides the foundational mental model students need before building any SwiftUI app. |
| *What's new in SwiftUI* | 2024 | New features including improved animations, custom containers, and mesh gradients. | Keeps your curriculum current with the latest SwiftUI capabilities students can leverage. |
| *Demystify SwiftUI* | 2021 | Deep dive into how SwiftUI's identity, lifetime, and dependency systems work under the hood. | Helps teachers explain *why* SwiftUI behaves the way it does, reducing student confusion. |
| *SwiftUI on all devices* | 2024 | Building apps that adapt across iPhone, iPad, Mac, Apple Watch, and Apple TV. | Shows students how one codebase runs everywhere -- a powerful motivator for learning SwiftUI. |
| *Migrate your MVVM architecture to SwiftUI* | 2023 | Patterns for structuring SwiftUI apps using MVVM and the Observation framework. | Teaches clean architecture patterns that students should learn early in their development journey. |

## Recommended Viewing Order for New Teachers

If you are new to teaching Apple development, watch these sessions in this order:

1. *A Swift Tour: Explore Swift's features and design* (2024) -- Understand the language.
2. *SwiftUI essentials* (2024) -- Learn the UI framework.
3. *Develop in Swift* (2024) -- See Apple's educational approach.
4. *What's new in Swift* (2024) -- Stay current.
5. *Introducing visionOS* (2023) -- Get inspired about the future.

## How to Access

- All WWDC sessions are free to watch at [developer.apple.com/videos](https://developer.apple.com/videos/).
- Sessions include transcripts, code samples, and related documentation.
- The Apple Developer app (free on iOS/macOS) allows offline viewing.
