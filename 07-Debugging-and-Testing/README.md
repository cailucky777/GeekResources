# 07 — Debugging and Testing

> The single most underrated skill for new programmers: **what to do when your code breaks.**

Most students write code assuming it will work. Real programmers assume it won't, and build the habits to find and fix problems quickly. This unit teaches both.

---

## What You'll Learn

- How to read and interpret error messages (the scariest-looking ones are usually the easiest).
- How to use Xcode's debugger: breakpoints, stepping, the variable inspector.
- How to use `print()`, `assert()`, and `#warning` effectively.
- How to write your first unit tests in Swift.
- How to isolate a bug (binary search, minimal reproduction).
